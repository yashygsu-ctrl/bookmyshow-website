# BookMyShow Frontend Clone

A pixel-perfect frontend clone of the popular entertainment ticketing platform, **BookMyShow**. This project replicates the user interface layout of the core landing page, showcasing modern web layout structures, clean semantic engineering, and interactive element designs.

🌐 **[View Live Website](https://yashygsu-ctrl.github.io/bookmyshow-website/)**

---

## 🚀 Key Features Implemented

*   **Responsive Navigation Bar:** Implemented a full-width header featuring logo integration, unified search constraints, location selectors, and call-to-action buttons.
*   **Dynamic Banner Slider Carousel:** Created the focal marketing carousel layout for seamless movie and event promotional highlights.
*   **Structured Entertainment Category Cards:** Modular grids cleanly presenting specialized cards for Movies, Live Events, Stream streaming options, Sports, and Activities.
*   **Hover States & Micro-interactions:** Fluid interactive states applied across image elements, buttons, and site anchor menus using CSS transitions.

---

## 🛠️ Tech Stack Used

*   **HTML5:** Structured semantic markup utilizing standard web hierarchy tags (`<header>`, `<nav>`, `<section>`, `<footer>`).
*   **CSS3:** Styled architectures implementing advanced styling mechanics, layout structures, typography properties, and smooth user hover states.
